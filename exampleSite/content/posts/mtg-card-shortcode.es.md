Usa el shortcode `card` cuando una entrada tenga que detenerse y mostrar una carta con claridad. Recibe el nombre de una carta de Magic y renderiza una vista previa con su imagen.

{{< card "Lightning Bolt" >}}

```
{{</* card "Lightning Bolt" */>}}
```
